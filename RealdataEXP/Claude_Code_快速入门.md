## 🔧 常见问题解决

### API连接失败
```bash
# 检查网络连接
curl -I https://open.bigmodel.cn/api/anthropic

# 验证API key
echo $ANTHROPIC_AUTH_TOKEN
```

### 会话中断
```bash
# 恢复最近会话
claude -c

# 查看所有可用会话
claude config list-sessions
```

### 性能优化
```bash
# 使用高速模式（如果可用）
export ANTHROPIC_MODEL="claude-sonnet-4-turbo"
claude
```
